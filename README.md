# BunnyEra Console

BunnyEra Console is the cross‑platform desktop interface of the BunnyEra ecosystem.  
It unifies AI agents, automation workflows, cloud services, and cross‑border tools into a single operating environment.

## Features
- Unified desktop console for BunnyEra OS
- Integrated AI agent panel
- Cloud service access and API routing
- System information and diagnostics
- Modular architecture for future extensions

## Tech Stack
- Electron
- Node.js
- Playwright
- SystemInformation
- Express

## Project Structure
- `/modules` — Core functional modules (Agent, Log, System)
- `main.js` — Electron main process
- `renderer.js` — UI logic
- `index.html` — Base UI layout
